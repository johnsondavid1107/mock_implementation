How Does Freshpaint Work?

🛠
  • Freshpaint automatically captures user actions across FreshHealth product and website
  • Freshpaint stores this data until FreshHealth is ready to use it
  • Freshpaint prevents all of the eighteen HIPAA identifiers from being shared with tools like Google Analytics while ensuring FreshHealth get all the reporting they need to measure performance
🏃

Action Steps

  • Test Freshpaint integration across different browser platforms to ensure data accuracy and compliance
  • Relay onboarding schedule
